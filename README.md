# Colin Shanafelt - Biography Page

This repository contains a professional HTML biography page for Colin Shanafelt.

## Setup

1. Place the profile images in the `assets/images/` directory:
   - `profile1.jpg` - Main profile photo
   - `profile2.jpg` - Secondary profile photo at desk

2. Open `index.html` in a web browser to view the biography page.

## Features

- Responsive design that works on desktop and mobile devices
- Embedded YouTube videos
- Interactive book previews using FlipHTML5
- Links to published works on Amazon
- Professional styling with modern CSS

## Hosting

To host this page online, you can use:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service

Simply upload all files maintaining the directory structure.
